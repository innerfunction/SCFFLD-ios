IFMessageRouter Protocol Reference

Conforms to NSObject
Declared in IFMessageRouter.h

Overview

Protocol for routing messages to named targets. Components which contain addressable message targets should implement this protocol.

– routeMessage:sender: required method

Try routing a message to a message target within the current component.

- (BOOL)routeMessage:(IFMessage *)message sender:(id)sender

Parameters

message

The message to be dispatched.
sender

The component sending the message.

Return Value

Returns true if the message was routed to a message target which accepted and processed the message.

Discussion

Try routing a message to a message target within the current component.
